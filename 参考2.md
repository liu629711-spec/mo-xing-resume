# 任务：进入电脑世界 —— 沉浸式个人网站

## 角色定位
你是叙事设计师 + 前端工程师 + 游戏UI收藏家。
不做"好看的网站"，做"让人想讲给朋友的体验"。

---

## 核心创意：「故障重启」

把个人网站设计成一台"被故障的复古电脑"。
访客不是浏览者，是**误打误撞进入这台电脑的人**。

---

## 叙事结构（完整故事线）

```
[第一章：黑暗中的光]        [第二章：系统启动]      [第三章：鼠标轨迹]
键盘在黑暗中发光           屏幕亮起，BIOS风格      鼠标出现，留下轨迹
访客意识自己在这里         桌面加载完成            跟随鼠标探索

         ↓                        ↓                      ↓

[第四章：经典游戏菜单]  →  [第五章：游戏内世界]  →  [第六章：通关结算]
点击打开经典游戏            每个游戏=一段履历         完成所有"游戏"后
但是游戏内容是履历          用游戏语言讲我的故事       解锁"通关成就"
```

## 第一章：黑暗中的光（入口）

### 场景设定
- 全黑屏幕，只有一把键盘的轮廓隐约可见
- 键盘按键有微弱的光渗出（青色/绿色CRT风格）
- 屏幕底部有一行字闪烁："按任意键开始..."

### 交互
- 访客按下任意键 → 键盘按键音效
- 屏幕开始闪烁/静电噪音
- 一段"系统启动"序列动画

### GSAP编排
```
timeline:
1. 静电噪音效果（0-0.5s）
2. 屏幕扫描线出现（0.5-1s）
3. "BIOS POST..." 文字逐字出现（1-2s）
4. "Booting..." 进度条（2-4s）
5. 屏幕爆发白光过渡到桌面（4-4.5s）
```

---

## 第二章：系统启动（桌面加载）

### 场景设定
复古Windows/Mac桌面风格，但有"故障艺术"元素：
- 图标排列有点歪斜
- 壁纸是像素风格的"我的照片"
- 任务栏显示时间、奇怪的系统信息

### 桌面图标（每个是一扇门）
```
📁  关于我.exe
📁  技能树.dat
📁  副本挑战.log
📁  成就殿堂.sav
📁  组队招募.url
📄  访客笔记.txt        ← 自动打开的备忘录
```

### 交互
- 图标hover：轻微抖动 + 名称tooltip
- 双击图标：窗口从图标位置"弹出"打开
- 窗口打开动画：经典的"窗口缩放"效果

---

## 第三章：鼠标轨迹（全局系统）

### 核心交互
- 自定义鼠标：一个小像素光标
- 移动时：留下逐渐消失的轨迹（像CRT残影）
- 点击时：经典的"点击涟漪"效果
- 双击时：两个涟漪重叠

### 彩蛋交互
- 把鼠标移到屏幕边缘：触发"屏幕翻角"效果（像翻书）
- 快速晃动鼠标：触发"幽灵鼠标"残影
- 长时间不动：进入"屏幕保护程序"（我的照片飘动）

## 第四章：经典游戏菜单（导航界面）

### 创意核心
当访客点击桌面图标，不是直接打开窗口，
而是先出现一个"游戏选择界面"，风格致敬经典游戏：

#### A. 技能树 —— 致敬《暗黑破坏神》技能盘
- 中央是我的人物立绘
- 周围环绕技能球，点击展开技能详情
- 连接线表示技能关联
- hover时技能球发光

#### B. 项目副本 —— 致敬《最终幻想》菜单
```
╔══════════════════════════════╗
║   ▶ 我的项目.ff7              ║
╠══════════════════════════════╣
║  📁 项目1：XX系统             ║
║  📁 项目2：XX活动             ║
║  📁 项目3：XX工具             ║
║  📁 项目4：XX增长             ║
╠══════════════════════════════╣
║  ▶ 返回桌面    ◀ 上一个  ▶下一个 ║
╚══════════════════════════════╝
```

#### C. 工作经历 —— 致敬《勇者斗恶龙》剧情日志
- 文字框风格，有"下一页"按钮
- 每段经历是一章剧情
- 有"HP/MP"风格的数据展示
- 重要成就有"升级"动画

#### D. 成就殿堂 —— 致敬《塞尔达》收集界面
- 成就徽章像塞尔达的神庙符文

- 已解锁的有光晕+动画
- 点击有"叮"音效反馈

## 第五章：游戏内世界（内容展示）

### 进入方式
点击"游戏菜单"中的选项 → 屏幕"切换频道"效果 → 进入游戏内世界

#### A. 副本挑战内景（项目详情）
**致敬：《空洞骑士》关卡风格**

视觉：
- 横向卷轴，关卡在前方展开
- 背景有深度视差（3-4层）
- 沿途有"存档点"（里程碑事件）

交互：
- 自动卷轴前进，或手动控制
- 点击存档点：展开详细描述
- 通关后：出现"BOSS击破"成就

内容翻译：
| 真实内容 | 游戏语言 |
|---------|---------|
| 用户增长100万 | BOSS血量100万 |
| 活动参与率 | 伤害输出率 |
| 项目上线 | 通关副本 |
| 团队协作 | 组队配合 |

#### B. 技能树内景（技能展示）
**致敬：《暗黑破坏神》技能树**

视觉：
- 中央是我的人物剪影
- 周围技能球环绕，按类型分颜色
- 点亮的技能有粒子效果连接

交互：
- hover技能球：显示技能详情
- 点击技能：播放"加点"动画
- 形成独特的"Build"展示

#### C. 成就殿堂内景（数据展示）
**致敬：《空洞骑士》梦境之主**

视觉：
- 成就像悬浮的"符文"
- 未解锁的符文黯淡+锁链
- 已解锁的符文发光+粒子环绕

交互：
- 点击符文：符文旋转展开详情
- 所有已解锁：触发"全成就"烟花

## 全局系统

### 屏幕效果（贯穿全程）
- 轻微的CRT扫描线
- 偶尔的静电噪音闪烁
- 屏幕边缘轻微弯曲（ vignette）
- 光标拖尾残影

### 任务栏
- 左：开始按钮（返回桌面）
- 中：当前位置breadcrumb
- 右：探索进度 + 隐藏成就数

### 快捷操作
- ESC：返回上一级/关闭窗口
- 数字键1-5：快速打开对应程序
- ~（Tab上方的键）：切换CRT滤镜强度

### 音效（可选增强）
- 键盘敲击：按下任意键
- 点击确认：经典Windows点击音
- 窗口打开：缩放弹出音效
- 成就解锁：游戏获得道具音
- 静电噪音：随机触发

## 技术实现

### GSAP技能应用清单
| 效果 | GSAP实现 |
|-----|---------|
| 开机序列 | `gsap.timeline()` |
| 图标hover抖动 | `gsap.to()` + 随机x/y |
| 窗口弹出 | `gsap.fromTo()` scale |
| 鼠标轨迹 | `gsap-motionpath` 跟随 |
| 扫描线 | CSS animation |
| CRT效果 | CSS filter + 叠加层 |
| 技能球旋转 | `gsap.to()` loop |
| 粒子效果 | `gsap-physics2d` |
