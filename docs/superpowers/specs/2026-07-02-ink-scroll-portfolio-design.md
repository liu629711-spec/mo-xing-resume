# 水墨长卷·山水漫游 —— 游戏运营求职简历网站设计规格

> 主题：东方水墨 + 生成艺术 + 金缮 + 书法粒子 + 四季流转
> 定位：游戏产品运营求职简历
> 交互：长卷轴横向滚动驱动 3D 水墨山水
> 技术栈：原生 HTML/CSS/JS + Three.js（重 3D）+ GSAP + Lenis + D3.js

---

## 1. 核心概念

把网站做成一台"3D 水墨长卷展卷机"。访客滚动即"展卷"，职业生涯在云雾山峦间徐徐浮现。每个板块是一处景致，每个特效都有叙事意义而非炫技。

**差异化定位**：

| 普通简历 | 本作品 |
|---------|--------|
| 滚动浏览 | 展卷漫游 |
| 静态罗列 | 景致叙事 |
| 信息表格 | 数据墨河 |
| 点击跳转 | 沿径前行 |

---

## 2. 整体架构

```
访客进入
  ↓
[开卷仪式] Loading：墨滴落纸 → 晕染扩散 → 聚成标题
  ↓
[长卷主体] 横向 3D 场景，摄像机沿 X 轴推进
  ↓ 滚动驱动 ScrollProgress (0→1)
  ↓
[七大景致] 沿卷轴排布，进入视口时触发该板块动画
  ↓
[收卷] 末尾"归舟渡口"，联系方式 + 致谢
```

### 核心机制

- 整个 3D 场景是一条沿 X 轴延伸的"画卷带"，长度约 70 个世界单位（每板块 10 单位）。
- 滚动进度 `ScrollProgress ∈ [0,1]` 映射到摄像机 X 位置和板块激活状态。
- Lenis 平滑滚动 + GSAP ScrollTrigger 驱动 Three.js 摄像机。
- 每个板块 = 一段独立 3D 场景片段 + 一层 HTML/Canvas 信息层（文字、图表、可读内容）。3D 是氛围，信息层是内容，二者通过 scroll 同步。

### 配色（基底 · 四季流转动态切换）

|------|------|------|--------|------|

| 夏 | 冷宣 #E8EDE8 | 焦墨 #0D0D0D | 金 #B89968 | 翠绿 #6B8E7F |

| 冬 | 雪宣 #F0F0F2 | 淡墨 #2A2A2A | 银 #A8A8B0 | 雪青 #B8C5D6 |

默认按访客本地月份切换（3-5 春 / 6-8 夏 / 9-11 秋 / 12-2 冬），右上角提供手动切换节气图标。

### 字体

- 标题：`Ma Shan Zheng` 或 `ZCOOL XiaoWei`（Google Fonts，书法感）
- 正文：`Noto Serif SC`（思源宋体）
- 数字/英文：`Cormorant Garamond`

### 材质语言

- **宣纸底**：全屏 Canvas 噪点纹理，模拟宣纸纤维，随滚动有细微位移。
- **水墨着色器**：Three.js ShaderMaterial，noise + 流体模拟做墨晕扩散，山峦地形用此材质。
- **金缮金**：金属 PBR + emissive，裂纹用 geometry 沿屏幕空间生成。
- **书法粒子**：标题字符预渲染为粒子点位（Canvas 取像素），GSAP 控制聚合/散开。

### 动效原则

- "墨韵"为主：扩散、晕染、流淌、笔触。
- "金缮"为断：板块切换的"破坏-修复"瞬间，制造节奏停顿。
- "四季"为底：全局氛围色温变化，不抢戏。
- 60fps 优先，着色器复杂度按设备降级。

## 4. 七大板块详细设计

### 板块 0：开卷仪式（Loading + 入场）

**景致**：空白宣纸，一滴墨从天而降。

- **Loading 序列**（GSAP timeline，约 4 秒）：
  1. 全屏宣纸底淡入，中央一滴墨从上方落下（重力加速）。
  2. 墨滴落纸瞬间，墨晕扩散（着色器 ripple），同步发出一声"滴"（Web Audio 合成，可关）。
  3. 墨晕继续扩散，浮现你的名字（书法字体，从墨晕中心淡出）。
  4. 墨晕边缘裂开金缮纹路，纹路延伸铺满屏幕，屏幕"碎裂"后金线修复，过渡到第一板块。
- **加载内容**：预加载 Three.js 场景、着色器、字体、四季贴图。进度以墨晕扩散比例同步显示。
- **跳过**：右上角"跳过开卷"链接。

### 板块 1：入山亭 · 个人介绍

**景致**：墨竹掩映的小亭，远处朦胧山影。

- **3D 场景**：左侧几竿墨竹（粒子笔触生成，随风微摆），右侧简笔亭子，背景远山雾气。
- **信息层**：
  - 大标题："入山" + 你的名字（书法粒子聚合）
  - 一句话定位（如"游戏产品运营 · 用数据讲故事的造梦人"）
  - 三行自我评价关键词，墨笔写下动画
  - 右下角小印章（名字篆刻样式，hover 微旋转）
- **交互**：竹叶随鼠标轻摆；标题字进入视口时墨点聚合。

### 板块 2：星罗棋盘 · 技能矩阵

**景致**：云海之上悬浮的发光棋盘，棋子是墨玉质感。

- **3D 场景**：云海平面（着色器流动），上方悬浮大棋盘（3D plane），棋子是球体，按技能类别分区。
- **信息层**：hover 棋子弹技能卡（技能名 + 熟练度 + 一句话）。
- **技能分类**（游戏运营向）：
  - 活动策划：版本活动、节日活动、联动策划
  - 数据分析：DAU/留存/ARPU/LTV、SQL、Excel、BI
  - 用户增长：拉新、召回、社群、KOL
  - 商业化：付费设计、ROI、礼包
  - 工具：Axure、Xmind、数据平台、PS
- **熟练度可视化**：棋子高度 = 熟练度，发光强度随熟练度。
- **交互**：拖拽旋转棋盘视角；点击棋子展开"我是怎么练成的"小故事。

### 板块 3：峰峦叠嶂 · 运营项目案例

**景致**：连绵山峰，每座山是一个项目。

- **3D 场景**：3-5 座水墨山峦沿 X 排列，山高 = 项目影响力，山腰云雾环绕。
- **信息层**：每座山前悬浮"山名牌"写项目名。点击山体进入"项目详情卷"。
- **项目详情卷**（全屏 overlay，"展卷"展开动画）：
  - 项目名 + 运营时间 + 角色
  - 核心数据三连卡（DAU 增长 / 留存提升 / 收入变化）
  - 职责描述
  - 关键举措列表
  - 关键成果图（可选）
  - 关闭按钮："收卷"
- **交互**：山体 hover 抬升 + 云雾散开；点击进入详情；详情内可滚动。

### 板块 4：墨河奔流 · 数据成果可视化

**景致**：一条墨河横贯，河中数据如墨点漂流。

- **3D 场景**：流动墨河（着色器流体），河面漂浮发光数据点。
- **信息层**：核心运营指标可视化（D3/Canvas，水墨风格）：
  - 大数字卡：累计运营用户数、总收入、活动场次
  - 折线图：某项目 DAU 增长曲线（墨笔绘制动画）
  - 环形图：用户来源构成
  - 柱状图：各活动 ROI 对比
- **交互**：滚动到此处图表逐个"墨笔绘制"；hover 数据点显示详情。
- **数据来源**：本地 JSON 配置，支持脱敏（百分比而非绝对值）。

### 板块 5：碑林 · 运营过的游戏作品墙

**景致**：一片矗立的墨碑，每块碑是一款游戏。

- **3D 场景**：3-6 块石碑排列，碑面贴游戏 icon/截图，碑顶题字。
- **信息层**：hover 碑体显示浮层（游戏名 · 类型 · 运营周期 · 角色）。
- **交互**：碑林视角随鼠标视差转动；点击碑体放大查看详情卡（含运营心得）。
- **NDA 处理**：未上线/NDA 项目用"无字碑"占位，hover 显示"因保密协议不便展示"。

### 板块 6：山径蜿蜒 · 经历时间轴

**景致**：一条蜿蜒上山的小径，节点是小亭或路碑。

- **3D 场景**：S 形路径（SVG path 投影到 3D 地面），节点是 3D 小亭，路径旁云雾。
- **信息层**：每节点 = 一段经历（公司/学校 · 职位 · 时间 · 一句话），沿路径排列。
- **交互**：滚动驱动"行旅墨点"沿路径前进，到达节点时节点点亮、信息卡淡入。
- **情绪标签**：每节点带小印章（🚀 启程 / 📈 成长 / 🎉 成就 / 💡 觉悟），盖章动画。
- **转折点**：关键节点路径分叉，显示"那段我选择了..."小注解。

### 板块 7：归舟渡口 · 联系方式

**景致**：江边渡口，一叶扁舟，远山夕阳。

- **3D 场景**：水面（着色器波纹），墨舟随波轻摇，远处夕阳（金缮金光晕）。
- **信息层**：
  - 大标题："归舟" + "期待与你同渡"
  - 联系方式：邮箱（点击复制）、微信（hover 显示二维码）、电话（可选）、LinkedIn/脉脉
  - 邀请文字（如"如果你也在做让人心动的游戏，聊聊？"）
- **交互**：邮箱点击复制 + toast"已折好信封"；微信 hover 墨舟上方浮现二维码；提交后水面荡开金涟漪。
- **彩蛋**：输入"面试"/"offer"等关键词，夕阳变金缮满屏，显示"期待加入"。

## 5. 全局特效系统

### 5.1 金缮过渡

板块切换（ScrollTrigger 进入/离开）时触发：

1. 屏幕空间生成一道金色裂纹（Canvas 2D 叠加层，分形噪声生成树状裂纹路径）。
2. 裂纹沿路径"生长"（GSAP path 动画，约 0.6s）。
3. 裂纹两侧画面轻微位移错位（CSS clip-path + transform）。
4. 金线沿裂纹"修复"填充（gradient 沿 path 描边，约 0.4s）。
5. 修复完成，新板块淡入。

实现位置：`js/effects/kintsugi.js`，监听 ScrollTrigger 事件。

### 5.2 书法粒子标题

每个板块大标题、个人名字使用：

1. 离屏 Canvas 用书法字体渲染目标文字，按网格采样像素点位（透明度 > 阈值的像素）。
2. 每个像素点生成一个 Three.js Points 粒子或 DOM 粒子（视性能）。
3. 初始状态：粒子随机散布在屏幕，缓慢漂浮。
4. 进入视口时：GSAP tween 所有粒子到目标点位，聚合为字（约 1.2s，elastic 缓动）。
5. 离开视口：粒子缓慢散开。

实现位置：`js/effects/calligraphy-particles.js`，输入文字 + 字体 + 颜色 → 返回粒子系统。

### 5.3 四季流转

- `js/core/seasons.js`：根据 `new Date().getMonth()` 决定当前季节，暴露 CSS 变量（`--paper`, `--ink`, `--gold`, `--accent`）和 Three.js uniform 值。
- 右上角节气切换器：4 个小图标，点击切换季节，全局 CSS 变量 + uniform 平滑过渡（GSAP tween，1s）。
- 季节影响：宣纸底色、墨色、金缮色、点缀色、粒子色调、水面/云雾色调。

### 5.4 滚动劫持与摄像机同步

- Lenis 初始化平滑滚动，`lenis.on('scroll', ...) => ScrollTrigger.update()`。
- 全局 `scrollProgress = lenis.scroll / lenis.limit` ∈ [0,1]。
- Three.js 摄像机：`camera.position.x = scrollProgress * 70`（GSAP quickTo 平滑）。
- 每个 ScrollTrigger section `start: 'top center'` 触发该板块 `activate()`，`end: 'bottom center'` 触发 `deactivate()`。

### 5.5 宣纸底纹

- 全屏 fixed Canvas（z-index 最底），用 Perlin noise 生成宣纸纤维纹理。
- 随滚动有细微位移（视差），随季节切换色调。
- 低分辨率生成 + CSS 放大，性能友好。

### 加载顺序

1. `core.css` + 字体（CDN 优先）。
2. `paper-bg.js` 启动宣纸底纹（纯 Canvas，不依赖 Three）。
3. Loading 序列启动，并行预加载 Three.js 场景、着色器、贴图、四季配置。
4. 资源就绪 → Loading 完成 → `scene.js` 启动渲染循环 → ScrollTrigger 激活。
5. 各 `sections/*.js` 注册自己的 ScrollTrigger，按需创建/销毁 3D 对象。

### 性能策略

- **按板块加载 3D 资源**：当前板块 ±1 范围内的 3D 对象才加入场景，远离的移除（池化复用）。
- **着色器复杂度分级**：检测 GPU 等级（`renderer.capabilities`），低配降级（减少 noise octaves、关闭流体模拟改用预渲染贴图）。
- **像素比限制**：`renderer.setPixelRatio(Math.min(devicePixelRatio, 2))`。
- **离屏 culling**：视锥外对象 `visible=false`。
- **图表懒绘制**：数据可视化只在进入视口时绘制一次。

## 8. 响应式与降级

### 桌面（>1024px）
完整 3D + 所有特效。

### 平板（768-1024px）
- Three.js 渲染分辨率降至 1x。
- 关闭书法粒子聚合（标题用纯字体淡入）。
- 金缮过渡保留但简化裂纹分支。

### 手机（<768px）
- 自动降级为"2D 水墨卷轴"模式：Three.js 场景替换为预渲染水墨背景图 + CSS 视差。
- 滚动改为纵向（移动端横向滚动反直觉）。
- 板块内容垂直堆叠，触控友好。
- 保留：宣纸底纹、四季配色、墨笔图表动画、金缮过渡（简化）。
- 移除：3D 山峦/棋盘/碑林/墨舟/书法粒子。

### 减弱动画（`prefers-reduced-motion`）
- 关闭金缮、书法粒子、墨滴落纸、摄像机推进。
- 内容直接淡入，滚动跳转无平滑。

## 9. 错误处理

- **WebGL 不支持**：检测 `WebGLRenderingContext`，不支持则直接走手机版 2D 降级路径。
- **资源加载失败**：字体/贴图加载失败时回退系统字体/纯色材质，不阻塞主流程。
- **JSON 数据格式错误**：`data/*.json` 解析失败时控制台告警，对应板块显示"内容更新中"占位，其余板块正常。
- **ScrollTrigger 与 Lenis 冲突**：使用官方推荐集成方式，`lenis.on('scroll', ScrollTrigger.update)` + `gsap.ticker.add` 驱动 lenis。
