# 任务：打造独一无二的沉浸式个人网站

## 角色定位
你是产品总监 + 创意技术总监 + 前端架构师的三位一体。不追求"好看"，追求**令人难忘的体验**。

---

## 核心设计理念

### 反常规设计原则
- ❌ 拒绝：标准单页滚动、固定导航栏、卡片式作品展示
- ✅ 追求：**空间叙事**、**非线性探索**、**游戏化交互**、**情绪曲线设计**

### 差异化定位
| 普通作品集 | 你的作品 |
|-----------|---------|
| 滚动浏览 | 空间漫游 |
| 静态展示 | 可交互叙事 |
| 信息罗列 | 情绪引导 |
| 点击跳转 | 无缝沉浸 |

## 网站架构：「数字分身」概念

把网站设计成用户的**数字孪生空间**，访客不是在"看简历"，而是在"探索一个人的数字宇宙"。

### 整体结构：三维空间导航
```

        │                │                │
        └────────────────┼────────────────┘
                         │
                    [联系奇点]
```

## 板块设计（每个都是独特体验）

### 1. 入口：「诞生星云」Loading + 入场
**概念**：用户从混沌中"诞生"进入你的数字宇宙

- **预加载**：粒子从屏幕中心坍缩成Logo，再爆发成星云
- **入场仪式**：
  - 访客输入一个昵称（可选，用于后续个性化）
  - 摄像头请求（可选，用于生成访客头像粒子）
  - 一段30秒的"引导灵"语音/文字（欢迎语 + 操作指引）
- **导航方式**：不是菜单，是**星图**——点击星球进入不同板块

**技术**：
- Three.js / Canvas 粒子系统
- GSAP `gsap.timeline()` 编排入场动画序列
- Web Audio API 生成环境音（可选）

---

### 2. 能力星系：「技能星座」可视化
**概念**：技能不是列表，是**可互动的星座图谱**

- **视觉**：深色宇宙背景，技能是发光的星星，连线成星座
- **交互**：
  - 鼠标靠近：星星放大，显示技能名称 + 熟练度
  - 点击星星：展开"技能故事"——一段文字讲述你如何掌握它
  - 拖拽旋转：3D球体旋转，从不同角度查看
- **动态数据**：
  - 实时GitHub贡献数据（API拉取）
  - 最近使用的技术栈（从提交记录分析）
  - "正在学习"的脉冲新星（闪烁提示）

**技术**：
- D3.js 或 Three.js 力导向图
- GSAP `gsap-motionpath` 连线动画
- 实时数据：GitHub API + 本地缓存

---

### 3. 作品虫洞：「项目穿梭」体验
**概念**：每个作品是一个**可进入的独立维度**

- **入口**：悬浮在宇宙中的"虫洞"（漩涡状传送门）
- **交互**：
  - 悬停：虫洞加速旋转，预览作品缩略图
  - 点击进入：全屏沉浸式项目展示
- **项目内部结构**（每个项目独立设计）：
  ```
  项目维度内部：
  ├── 入口大厅：项目概览 + 核心数据（用户量、技术栈）
  ├── 时间回廊：开发历程时间轴（可点击节点看详情）
  ├── 演示空间：可交互Demo（直接运行）
  ├── 媒体墙：图片/视频画廊（手势滑动）
  └── 出口：返回主宇宙或跳转到下一个项目
  ```
- **项目类型差异化**：
  - 工具类：直接嵌入可运行的工具
  - 视觉类：全屏图片 + 视差滚动
  - 视频类：影院模式播放
  - 代码类：代码高亮 + 架构图 + 实时Demo

**技术**：
- 每个项目独立HTML，通过`iframe`或动态加载
- GSAP `gsap-scrolltrigger` 视差效果
- 自定义光标：进入项目后光标变成"探索者"样式

---

### 4. 时间轴星轨：「人生漫游」叙事
**概念**：经历不是列表，是**可漫步的星轨**

- **视觉**：一条发光的螺旋轨道，每个节点是一颗行星
- **交互**：
  - 滚动或拖拽沿轨道前进
  - 每个节点：年份 + 事件 + 情绪标签（🚀 兴奋 / 😰 挑战 / 🎉 成就）
  - 点击节点：展开"记忆碎片"——当时的照片、代码片段、感悟文字
- **特殊节点**：
  - 转折点：轨道分叉，显示"如果当时选择另一条路"
  - 里程碑：行星爆炸成烟花，庆祝成就

**技术**：
- SVG路径动画 + GSAP `gsap-motionpath`
- 滚动劫持（Scroll Hijacking）控制轨道进度
- 粒子烟花效果

---

### 5. 联系奇点：「量子纠缠」连接
**概念**：建立联系是跨越时空的量子纠缠

- **视觉**：一个旋转的奇点（黑洞效果）
- **交互**：
  - 输入框悬浮在奇点周围
  - 每输入一个字符，奇点脉动一次
  - 提交后：信息被"吸入"奇点，然后奇点爆发成感谢文字 + 社交链接
- **社交链接**：不是图标，是**卫星**——围绕奇点旋转，hover时飞近
- **彩蛋**：输入特定关键词（如"招聘"、"合作"），奇点变色并显示不同欢迎语

**技术**：
- Canvas 黑洞扭曲效果
- GSAP `gsap-physics2d` 或自定义物理
- 表单验证 + 提交动画

## 全局系统

### 导航系统：「星图」
- 固定右下角：迷你星图，显示当前位置
- 点击星图：展开全屏导航，快速跳跃到任意板块
- 快捷键：`1-5` 快速跳转

### 进度系统：「探索度」
- 右上角显示探索进度百分比
- 首次访问：引导发现隐藏内容
- 成就徽章："星座收集者"（查看所有技能）、"虫洞穿梭者"（访问所有项目）等

### 个性化系统
- 记住访客昵称，后续访问显示"欢迎回来，[昵称]"
- 根据访问深度调整引导语（新手/回归用户不同）

### 性能与适配
- 低端设备：自动降级为2D版本，保留核心体验
- 移动端：手势操作替代鼠标悬停，简化3D效果
- 加载策略：分块加载，优先首屏

## 技术栈

### 必用（本地 skills）
- `gsap-core`：所有动画核心
- `gsap-scrolltrigger`：滚动驱动动画
- `gsap-motionpath`：路径动画
- `gsap-splittext`：文字特效
- `gsap-physics2d`：物理效果（可选）

### 可用（线上资源）
- **Three.js**：3D场景、粒子系统
- **D3.js**：数据可视化
- **Lenis**：平滑滚动
- **Splitting.js**：文字拆分（替代GSAP SplitText）
- **Barba.js**：页面过渡（如需要多页面）

### 可选增强
- **WebGL**：自定义着色器效果
- **Web Audio API**：环境音效
- **TensorFlow.js**：摄像头交互（高级）
